# Choose Option Game

A simple interactive web application built with React that helps groups make decisions by combining user preferences and randomness.

## Overview

This project allows multiple participants to input their preferred options in order of priority. Based on these inputs, the application determines a result using a mix of matching logic and random selection.

The main goal of this project is to demonstrate fundamental React concepts such as state management, component composition, and handling user interactions.

## Features

- Dynamic number of participants  
- Step-by-step input flow for each participant  
- Support for prioritized options  
- Result calculation based on shared preferences or random fallback  
- Language switching (i18n support)

## Tech Stack

- React  
- TypeScript  
- i18next  

## Purpose

This project was created as part of my learning process in React, focusing on building a clean and functional user flow while keeping the implementation simple and readable.
